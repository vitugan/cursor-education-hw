1. Checking the Status

$ git status

2. adding changes

$ git add . //all changes $ git add file.txt //add file $ git add '*.txt' //batch add by mask

3. Fix changes (Commiting)

$ git commit -m "Add cute octocat story"

4. Check history

$ git log

5. Add remote repository

$ git remote add origin https://github.com/try-git/try_git.git

6. Push changes to remote repository

$ git push -u origin master

7. Pull remote repository

git pull origin master

8. Check the diffrences from last commit

$ git diff HEAD

9. Check the diffrence after pull.

$ git diff --staged

10. Reset the staged file (octodog.txt)

$ git reset octofamily/octodog.txt

11. Then files can be changed back to how they were at the last commit by using the command

git checkout -- target_file

12. Create branch

$ git branch new_feature

13. To see local branches

$ git branch

14. Switch between branches

$ git checkout ‹branch›

15 git rm command which will not only remove the actual files from disk, but will also stage the removal of the files for us.

$ git rm '*.txt'

16. To merge master branche with feature_branch:

$ git merge ‹feature_branch›

17. After merge branches we can delete ‹feature_branch›:

$ git branch -d ‹feature_branch›
